Introdução

O que é a PrestaShop e-commerce?

PrestaShop é uma solução de comércio eletrônico que tem um núcleo básico grátis e de código aberto. Ele suporta portais de pagamento como o Google checkout, paypal ou pagamentos pro via APIs.

PrestaShop está sob licença Open Software License e foi lançado oficialmente em Agosto de 2007 para empresas de pequeno e médio-porte. O programa, que é baseado no motor de template Smarty, é usado em milhares de lojas no mundo todo.

O que é um template do PrestaShop?

Template do PrestaShop é um tema para sua solução de comércio PrestaShop. Isso te possibilita uma grande variedade de oportunidades de personalizar seu site e permite que você crie uma loja online que atende as suas necessidades.

Estrutura de arquivos

The template package you have downloaded consists of several folders. Let's see what each folder contains:

  • screenshots - onde tem imagens de como seu tema se parecerá. Não é indicado para colocar no servidor de produção.
  • sources.zip - contém os arquivos fonte do template
    • sources
      • psd - com os arquivos fonte do Adobe Photoshop usados na criação do tema (.psd).
  • theme - contém os arquivos do tema do PrestaShop
    • manual_install - arquivos para instalação manual
      • img - imagens para instalação manual
      • modules - módulos para instalação manual
      • theme#### - arquivos do tema para instalação manual
      • dump.sql - dados de amostra para instalação manual
    • sample_data - contém dados de amostra para instalação com o módulo sampledatainstall
    • themeinstallator - contém um arquivo para instalação automática do tema
      • theme####.zip - arquivo do tema. Contém todos os arquivos do tema. É instalado usando o método padrão de instalação do PrestaShop.
  • fonts_info.txt - arquivo de texto com links para baixar das fontes personalizadas usadas pelo tema template
  • info.txt - instruções para extrair arquivos de código fonte

Primeiros passos

Peparando-se

Antes de prosseguir com a instalação de sua loja Prestashop, verifique se você está preparado. Siga os passos abaixo:

Programas necessários

Para trabalhar com templates do Magento, recomendamos baixar todas as aplicações necessárias. Você pode verificar a lista de software necessário na visualização do template.
Os requerimentos variam entre templates, mas para a maioria deles você vai precisar de:

  1. Primeiro, você vai precisar do WinZip 9+ (para Windows) ou do Stuffit Expander 10+ (para Mac) para extrair o conteúdo protegido por senha do arquivo sources_#########.zip.
  2. Você também deve precisar do Adobe Photoshop. Ele é usado para editar arquivos .PSD, e é necessário se você quiser editar gráficos e imagens do template.
  3. Para editar arquivos de código fonte, você vai precisar de um editor de códigos como Adobe Dreamweaver, Notepad++, Sublime Text, etc.
  4. Para subir arquivos para o servidor de hospedagem você vai precisar de um gerenciador de FTP como Total Commander, FileZilla, CuteFTP, etc.

Requisitos do servidor

Verifique se seu serviço de hospedagem suporta os seguintes requisitos:

  1. Sistema: Unix, Linux ou Windows. Unix é altamente recomendável.
  2. Servidor: suporta Apache Web server 1.3 ou posterior, Microsoft IIS Web server 6.0 ou posterior e nginx 1.0 ou posterior.
  3. PHP 5.2* ou posterior.
  4. MySQL 5.0 ou posterior.
  5. No mínimo 32 Mb de RAM no seu servidor (64 Mb é mais confortável, quanto mais memória melhor).
PrestaShop vai funcionar com o PHP 5.1 ou posterior, mas versões abaixo do 5.2 tem bugs que é podem impedir algumas funções de operarem como esperado (como fusos horários inválidos).

Usando um servidor local

você pode rodar o PrestaShop localmente no seu computador usando um servidor local. Para criar um servidor local, use os aplicativos próprios para isso, como WAMP, AppServ, MAMP etc. Qualquer um desses pode ser facilmente instalado como qualquer outro programa e usá-lo para rodar o PrestaShop.

Dê uma olhada nos tutoriais abaixo para aprender a configurar um ambiente de desenvolvimento local:

  • How to install AppServ web development environment
  • How to install WAMP web development environment
  • How to install XAMP web development environment

Baixando

Para instalar o PrestaShop, você precisa baixar o pacote do site oficial em www.prestashop.com/en/downloads

Preencha seu nome e email nos campos da caixa de download e clique no botão 'Download'.

Certifique-se de que sua versão do PrestaShop seja compatível com o seu tema. Todas as versões do PrestaShop estão disponíveis em SourceForge.

Extraindo os arquivos

Após baixar o pacote de instalação, você precisa extrair os documentos e pastas do arquivo.

Isso pode ser feito com qualquer gerenciador de arquivos, como WinZip(PC), StuffitExpander(MAC), etc.

Dê uma olhada nos tutoriais abaixo se encontrar dificuldades em extrair esses arquivos com WinZip(PC) e StuffitExpander(MAC).

Aliás: se o seu painel de controle da hospedagem tiver a opção de 'Extrair' arquivos, você pode usá-la para extrair os arquivos diretamente no seu servidor onde o está o arquivo .zip que você enviou.

Enviando os arquivos base do PrestaShop

Se você extraiu os arquivos localmente no seu computador, você vai precisar enviá-los para o servidor de hospedagem.

Você pode fazer isso usando o gerenciador de arquivos do seu painel de hospedagem ou usando um Gerenciador de FTP como o FileZilla, TotalCommander, CuteFTP, etc.

Você pode dar uma olhada num tutorial detalhado em how to upload files to server using FTP manager e how to upload files to a server using cPanel (WebHost Manager, WHM).

Instalando o PrestaShop

Quando terminar com o envio dos arquivos, você pode começar a instalação do PrestaShop. Abra seu navegador e digite 'nome do domínio/caminho para o diretório do PrestaShop no servidor' na sua barra de endereços, depois disso recarregue a página atual. Você deve ver a tela de boas vindas. Siga as instruções abaixo para instalar o PrestaShop

Passo 1: Assistente de instalação (Escolha seu idioma)

Escolha o idioma de instalação

Se liga: o idioma da instalação não vai afetar o idioma da sua loja. Por padrão, o PrestaShop é configurado em inglês. Você pode instalar pacotes de localização adicionais para mudar o idioma da loja.

Quando terminar, clique em 'Próximo (Next)'.

Passo 2: Assistente de Instalação (Acordo de Licenças)

Leia o acordo de licenças.

Quando terminar, clique em 'Próximo (Next)'.

Passo 3: System Compatibility

Check the PHP settings and directory permissions. In case of any errors, please contact your hosting provider for further assistance.

If everything is correct, clique em 'Próximo (Next)'.

Passo 4: Configuração da Loja

Nesse passo você precisa preencher configurações da loja como nome, atividade, país, fuso horário, etc.

Todas essas configurações podem ser alteradas mais tarde no painel administrativo do PrestaShop.

Passo 5: Configurações do Sistema

Se você já tiver um banco de dados MySQL, preencha os detalhes do banco nos campos abaixo.

Do contrário, você pode pegar os detalhes do seu banco de dados do seu painel administrativo no seu servidor de hospedagem, ou entrando em contato com o provedor de hospedagem.

Quando terminar, clique em 'Próximo (Next)'.

Passo 6: Instalação terminada

Esse é o último passo da instalação

Por razões de segurança, você precisa apagar o diretório /install/ no seu servidor de hospedagem.
À partir da versão 1.5.4, o PrestaShop cuida de renomear a pasta administrativa para você: se torna a palavra "admin" acrescida de 4 números aleatórios, por exemplo "admin8814". Essa mudança no nome acontece quando você acessa a pasta admin pela primeira vez: o PrestaShop renomeia a pasta e redireciona seu navegador para a pasta correta.

Faça login no escritório do PrestaShop indo até a sua pasta "admin" recém renomeada e comece a preencher seu catálogo com dados dos seus produtos.

Se encontrar qualquer dificuldade, entre em official PrestaShop installation manual.

Instalação do template

1. Importar o tema

Nota: esse método não providencia dados de demonstração instalados.

1. Preparação do template.

Antes de prosseguir, certifique-se de que você já baixou o seu template e extraiu os arquivos do pacote do template. Você pode dar uma olhada em tutoriais sobre descompactação/extração de arquivos aqui: WinZip and StuffitExpander.

Quando os arquivos são extraidos, você pode prosseguir com o envio do seu template para o servidor.

2. Instalação.

Entre no seu painel administrativo do PrestaShop e navegue até Preferências (Preferences) -> Temas (Themes). No topo da página, clique em Adicionar novo tema (Add new theme).

Clique em Adicionar arquivo (Add file) para importar do seu computador. Navegue até a pasta onde você descompactou os arquivos do template e abra a pasta themeinstallator.

Escolha o arquivo .zip do seu tema e clique em Salvar (Save).

O processo de envio será iniciado. Você verá uma mensagem de que o tema foi enviado com sucesso. O preview do seu tema vai aparecer na lista de temas instalados.

Para ativar o tema, passe o a seta do mouse em cima do ícone de preview do tema e clique em Usar este tema (Use this theme).

Você deve ver uma tela com informações dos módulos usados no seu tema e o número de módulos que estão desligados e ligados. Clique em Salvar (Save) para prosseguir.

Depois disso, você verá uma mensagem com a ativação do tema: o tema foi instalado com sucesso, imagens serão geradas novamente para caber nas dimensões do tema. Para terminar a instalação clique em Terminar (Finish).

2. Instalação automática com dados de amostra (Instalador de demonstração)

Após instalar os dados de amostra, você terá a loja do PrestaShop idêntica ao demo que você viu ao comprar o template. Ela terá todos os produtos de demonstração, categorias, módulos e configurações da loja.

Não use esses dados de amostra num site que está no ar! Isso vai substituir todos os produtos e configurações da sua loja.

Para instalar os dados de amostra usando o módulo instalador de dados de amostra, siga as instruções do passo 1 (Theme import) desse manual. Então, no painel administrativo de sua loja, você verá o módulo instalador de dados de amostra (Sample data installer module).

Ao usar este módulo você pode facilmente instalar dados de amostra na loja em poucos passos simples. Faça o seguinte:

  1. No menu do painel administrativo clique em "Instalar Loja de Demonstração (Install Demo Store)".

  2. Marque e envie os dados de amostra do diretório 'sample_data' no pacote do template. Você pode fazer isso usando o botão "Procurar arquivos locais (Browse local files)" ou arrastando os dados de amostra para dentro da janela do seu navegador.

  3. Espere terminar o envio.

  4. Então clique em "Continuar Instalação (Continue Install)".

  5. O instalador vai configurar sua loja e os outros módulos. Isso pode levar um tempo.

  6. Quando os dados de amostra forem instalados corretamente, você precisa fazer mais uma coisa - regerar as imagens. Clique em "Ir para a página de regeração (Go To Regenerate Page)".

  7. Para regerar as imagens, clique em "Regerar miniaturas (Regenerate thumbnails)" button.

Parabéns, sua loja de demonstração está instalada.

3. Instalação manual com dados de amostra

Depois de instalar os dados de amostra, você terá uma loja PrestaShop igual à demonstração do seu template. Ela terá os mesmos produtos de amostra, configurações de loja e módulos, etc.

Não use dados de amostra em sites que estão no ar! Isso vai apagar todos os produtos e configurações da sua loja.

1. Enviar arquivos do tema

Para enviar o tema para o seu servidor:

  1. Em primeiro lugar, abra o pacote do template.
  2. Vá para a pasta /theme/manual_install/.
  3. Envie as pastas /img/, /modules/ e /themes/ para dentro do diretório raiz do PrestaShop.

Você pode ver um tutorial em vídeo com mais detalhes em how to upload files to server using FTP manager e em how to upload files to a server using cPanel (WebHost Manager, WHM).

2. Instalar dados de amostra

Os dados de amostra vêm na forma de um arquivo SQL. Está em theme/manual_install/ e se chama dump.sql.

Esse arquivo de dump pode ser instalado através da ferramenta phpMyAdmin ou alguma outra ferramenta de gerenciamento de banco de dados no seu painel de controle de hospedagem. Esse tutorial detalhado pode te ajudar: how to install SQL files.

Antes de prosseguir, faça um backup do seu banco de dados. Aqui tem um tutorial de como fazer isso: how to back up a database.

Certifique-se de que sua versão do PrestaShop é igual aos requisitos de template. Do contrário, o banco de dados será comprometido.

Quando terminar de instalar o arquivo dump.sql, você vai precisar ativar o tema.

Do painel administrativo do PrestaShop, vá para Preferências (Preferences) > Temas (Themes), escolha seu tema na seção Temas (Themes) e clique em Salvar (Save).

3. Adicionar idioma à loja online.

Depois de instalar o tema com suporte à vários idiomas, você poderá adicionar todos os idiomas que vieram junto. (Inglês, Francês, Espanhol, Alemão, Russo) à plataforma. It can be done this way:

Vá para 'Localização (Localization) > Tradução (Translations)' no painel administrativo, encontre 'ADICIONAR/ATUALIZAR IDIOMA (ADD/UPDATE A LANGUAGE)' e adicione idiomas escolhendo no menu de escolha (1) e clicando no botão 'Adicionar ou atualizar um idioma (Add or update a language)' button (2).

Se não quiser ter algum idioma, você pode apagá-los (2) ou desativá-los (1) no menu 'Localização (Localization) > Tradução (Translations)' menu.

4. Como instalar o template em um site já existente

Você pode usar esse método se você já tem um site PrestaShop e quer apenas instalar um novo tema.

1. Enviar arquivos e ativar o tema

  1. Abra o pacote do template.
  2. Vá para a pasta /theme/manual_install/.
  3. Envie as pastas /img/, /modules/ and /themes/ para dentro do diretório raiz do PrestaShop.

    Você pode ver um tutorial em vídeo com mais detalhes em how to upload files to server using FTP manager e em how to upload files to a server using cPanel (WebHost Manager, WHM).

  4. No painel administrativo do PrestaShop navegue até Preferências (Preferences) > Temas (Themes), escolha o tema que você instalou (você também pode usar o tema habilitado, isso não ccausará problemas). Clique em Configurações avançadas (Advanced settings).

  5. Na tela que aparecer, escolha a pasta com seu novo tema e clique em Salvar (Save).

2. Instalar módulos

Cada tema vem com alguns módulos específicos para configurar. Se quiser que os módulos fique iguais ao site de demonstração, você vai precisar configurá-los. Todos os módulos podem ser configurados através do painel administrativo do PrestaShop.

1. No painel, escolha Módulos (Modules) > Módulos (Modules) no menu principal.

Na página de módulos, você pode ver uma lista de módulos PrestaShop disponíveis. Talvez você precise instalar alguns deles.

Para instalar um módulo, localize-o na lista e clique no botão 'Instalar (Install)'.

Os links dos módulos servem para ativar/desativar suas funcionalidades (clicando em Habilitar (Enable)|Desabilitar (Disable)), redefinir suas configurações para as configurações padrão, (Resetar (Reset)), apagar um módulo (Deletar (Delete)) ou configurá-los (Configurar (Configure)).

2. Após ativar os módulos necessários, você precisa definir as posições corretas deles.

No paienl, vá para Módulos (Modules) > Posições (Positions) no menu superior. Para mudar a posição de um módulo, clique e arraste até a posição desejada na lista de módulos.

3. Configurar imagens

No PrestaShop você pode usar tamanhos pré-definidos para as imagens de produtos, de categorias, fabricantes, etc.

Por exemplo, imagens com o tamanho 'home' em geral são para imagens de produto em listagens de categorias, 'large' para a imagem principal do produto na página de informações dele.

Para configurar o tamanho da imagem do produto, abra o seu painel do Prestashop, vá para Preferências (Preferences) > Imagens (Images) e mude os tamanhos das imagens como está descrito na página de Configurações (Configuration Page) no site de amostra do tema.

Tutoriais básicos

1. Como trocar o nome da loja

Para trocar o título da sua loja, abra o painel e vá para Preferências (Preferences) > Contatos da Loja (Store Contacts), na seção Detalhes de contato (Contact details).

No campo Nome da loja (Shop name), digite o nome de sua loja e clique em Salvar (Save) para alterá-lo.

2. Como alterar o slider

Para alterar as imagens do slider, encontre o módulo Slider de imagens para sua homepage (Image slider for your homepage) na parte de Módulos (Modules) de sua loja. Deve ficar na seção de módulos Fachada (Front Office Features). Clique em Configurar (Configure).

Na página de configurações do módulo, mude as opções do slider (tamanho, velocidade, etc.) e altere os slides (imagens, URLs, descrição, etc.).

Módulos e extensões personalizadas

Smart Blog

Smart blog - uma série de módulos que possibilitam a criação de um blog na sua loja. Essa série consiste dos seguintes módulos: Smart Blog, Feed de RSS do Smart Blog, Arquivo do Smart Blog, Categorias do Smart Blog, Smart Blog Home Latest (últimas notícias), Smart Blog comentários recentes, Smart Blog módulo do AddThis, Smart Blog Posts populares, Smart Blog Posts recentes, Smart Blog Related Post, Smart Blog busca, Smart Blog Tags. It also has its hooks where the blog modules can appear on the site, they can also show up in the standard hooks (only columns).

Smart Blog

Esse é o módulo principal que contém muitas configurações para a funcionalidade do blog. Depois de instalado, a aba de menu do Blog vai aparecer no menu principal de administração, onde você pode incluir/remover/alterar os posts/categorias ou selecionar o tipo de imagens e seus tamanhos usados nos posts (tipos e dimensões das imagens de post são diferentes das imagens de produto de sua loja). Para adicionar configurações de aparência ao módulo, navegue até Smart Blog (Modules->Front Office Features->Smart Blog) e adicione todos os parâmetros.

Descrição dos parâmetros:

  1. Meta Title - o título do seu blog que vai aparecer em todas as páginas.
  2. Meta Keyword - palavras-chave que serão usadas em ferramentas de busca para melhorar o SEO (Search Engine Optimization).
  3. Meta Description - descrição do seu blog usada por ferramentas de busca para melhor desempenho de SEO.
  4. Main Blog Url - o endereço do seu blog.
  5. Use .html with Friendly Url - ativa/desativa adicionar .html na barra de endereço para páginas utilizando URL amigável.
  6. Number of posts per page - número de posts que aparecem em cada página (categoria/arquivo, etc.).
  7. Auto accepted comment - ativa/desativa os comentários de usuário. Quando ativado, comentário serão incluídos sem moderação prévia.
  8. Enable Captcha - ativa/desativa Captcha em campos de comentário nas páginas de produtos.
  9. Enable Comment - ativa/desativa comentários de usuários.
  10. Show Author Name - ativa/desativa mostrar o autor dos posts em páginas do blog e páginas avulsas.
  11. Show Post Viewed - ativa/desativa mostrador de quanos usuários visualizaram o conteúdo. Esse número aparece nas páginas do blog e em páginas avulsas.
  12. Show Author Name Style - muda o formato do nome de autor.
  13. AVATAR Image: - adiciona imagens para todos os usuários.
  14. Show No Image - ativa/desativa a imagem. Nenhuma imagem é colocada se não houver imagem para o post/categoria, etc.
  15. Show Category - ativa/desativa a imagem e descrição de categoria nas páginas de categoria do blog.
  16. Blog Page Column Setting - configura a aparência de módulos relacionados ao blog. Em geral requer configurações adicionais, portanto é recomendável que você instale módulos adicionais de forma similar aos outros módulos do PrestaShop.
  17. Custom CSS - você pode incluir estilos adicionais neste campo. Tenha cuidado ao usar classes que já são utilizadas no tema, já que podem haver conflitos.
  18. Delete Old Thumblr - é usado para regerar imagens do blog se você alterar as dimensões delas em opções anteriores ou se um formato novo de imagem for adicionado. Para regerar, escolha Sim (Yes) e clique em Regerar (Regenerate).
APENAS APÓS ESTE MÓDULO outros módulos relacionados ao blog podem ser instalados. O processo de remoção é o contrário: PRIMEIRO, REMOVA OS MÓDULOS RELACIONADOS e depois apague o Smart Blog.
RSS Smart Blog Feed

O módulo que gera feeds de notícias do seu blog.

Tem diversas configurações no painel do admin:

  1. Update Period - permite mostrar posts de períodos específicos (hora/dia/semana/mês/ano).
  2. Update Frequency - permite escolher a frequência de atualização do Feed (1 semana, 2 horas, etc.).

No topo tem um caminho de arquivo onde esses arquivos de notícias serão gerados. Ele consiste de endereço do site + /modules/smartblogfeed/rss.php.

Arquivo do Smart Blog

Esse bloco aparece nas colunas laterais e é usado para organizar posts por período de tempo (mês).

Este módulo não possui parâmetros de configuração.

Categorias do Smart Blog

This block shows up in the left/right column and is used to sort posts by category.

Este módulo não possui parâmetros de configuração.

Smart Blog Home Latest Posts

Esse bloco mostra apenas no gancho homepagecontent e é usado para mostrar as últimas postagens na sua home page.

Configurações:

  1. Number of posts to display in Latest News - quantidade de postagens que vão aparecer na sua home page.
Smart Blog comentários recentes

Esse bloco aparece nas colunas laterais e é usado para mostrar os últimos comentários adicionados em posts do blog.

Configurações:

  1. Number of Comments to Show - quantidade de comentários para mostrar no bloco.
Smart Blog módulo do AddThis

Esse bloco aparece na página do post e permite compartilhá-la nas redes sociais.

Este módulo não possui parâmetros de configuração.

Smart Blog Posts populares

Esse bloco aparece nas colunas laterais e é usado para mostrar os posts mais populares/visitados.

Settings:

  1. Number of popular Posts to Show - quantidade de posts populares para mostrar no bloco.
Smart Blog Posts recentes

Esse bloco aparece nas colunas laterais e é usado para mostrar os posts mais recentes.

Settings:

  1. Show Number Of Recent Posts - quantidade de posts recentes mostrados no bloco.
Smart Blog Posts relacionados

Esse bloco aparece na página do post e é usado para mostrar posts relacionados.

Settings:

  1. Show Number Of Related Posts - quantidade de posts relacionados para mostrar no bloco.
Smart Blog busca

Esse bloco aparece nas colunas laterais e é um campo de busca para o blog.

Este módulo não possui parâmetros de configuração.

Smart Blog Tag

Esse bloco aparece nas barras laterais e é usado para mostrar as palavras-chave usadas no blog.

Settings:

  1. Number of Tag to Show - quantidade de palavras chave mostradas no bloco.

Media Parallax Module

O que é o efeito parallax?

O Parallax é uma técnica de rolagem especial em gráficos de computador, onde as imagens de fundo se movem mais devagar do que as imagens na frente, criando a ilusão de profunidade em um video game 2D e adicionando imersão. Web designers começaram a incorporar o parallax em 2011, usando HTML5 e CSS3. Sites com parallax estão se tornando uma estratégia popular, já que é uma forma simples de abraçar a fluidez da web. Além disso, usa-se o fundo em parallax como uma ferramenta para engajar usuários e aumentar a experiência do usuário no site. Rolagem em parallax permite a criação de experiências de usuários dinâmicas e interativas com o potencial de um apelo visual.

Esse módulo adiciona um efeito de parallax sem exigir habilidades ou esforços seus. É possível adicionar tanto imagens quanto vídeos.

Configurações: (apenas para o Google Chrome)

  1. Enable smooth scroll for Google Chrome browser - ativa efeito adicional.
  2. Smooth scroll time - duração do efeito da rolagem suave.
  3. Smooth scroll distance - distância do efeito da rolagem suave.

Adicionando o Módulo Media parallax

  1. É preciso adicionar o id ou a classe no conteúdo de sua página HTML. Você também pode usar uma tag única, esse elemento não deve ser repetido.
  2. Crie a imagem do tamanho necessário(de preferência criar uma imagem maior do que o bloco onde vai entrar o efeito). No caso de você quiser adicionar o vídeo, você vai preparar 3 formatos: .mp4,.webm e .ogv, e também adicionar a imagem de amostra do mesmo tamanho (.png, .jpg e .gif)

  3. Clique em Adicionar contêiner (Add container) e depois em Adicionar novo bloco (Add new block).

  4. Na tela que apareceu, inclua todas as configurações necessárias.

    Configurações:

    1. Selector - elemento onde o efeito será adicionado (id ou tag única).
    2. Parallax type - tipo de efeito utilizado (imagem ou vídeo).
    3. Media files - envie todos os arquivos necessários.
    4. Clique em Salvar (Save).

Personalizando o Módulo Media parallax

  1. Na lista de blocos de Parallax, encontre o bloco necessário e mude suas configurações.

Removendo o Módulo Media parallax

  1. Na lista de blocos, encontre e clique em Apagar (Delete).

Video Background Module

Esse módulo permite que você inclua um vídeo no fundo de qualquer bloco de sua página HTML (de preferência com id ou tag única).

Para inlcuir um vídeo de fundo, siga os passos abaixo:

  1. Envie os arquivos de vídeo e imagens de amostra para modules/tmvideobackground/video (ou qualquer outro diretório no servidor) via FTP. O vídeo deve ser enviado em 3 formatos: .mp4, .ogv, .webm, todos os arquivos tendo o mesmo nome (ocean.mp4, ocean.ogv, ocean.webm). Imagens de amostra devem também estar em 3 formatos: .jpg, .gif, .png e ter os mesmos nomes (ocean.jpg, ocean.gif, ocean.png).
  2. No seu painel administrativo, abra o Módulo para fundo com vídeo (Módulos (Modules)->Fachada (Front Office Features)->Módulo para fundo com vídeo) inclua as configurações necessárias.

    Configurações:

    1. Video selector - elemento da página HTML onde vai entrar o fundo com vídeo (id ou tag única, como body, header, footer, etc.).
    2. Video path - caminho para os arquivos do vídeo (o caminho deve terminar com /), os arquivos podem ser ficar em qualquer pasta no servidor, mas recomendamos que você envie-os para a pasta modules/tmvideobackground/video/.
    3. Video files name - nome dos arquivos de vídeo. Só o nome (não precisa escrever o formato. Por exemplo, "ocean").
    4. Clique em Salvar (Save).
Para mostrar o vídeo, não pode haver qualquer outro fundo adicionado no elemento.

TM HTML Content

Esse módulo permite incluir qualquer conteúdo (imagem, HTML, links, etc.) nas seguintes posições: displayNav, home, top, topColumn, left e right (displayLeftColumn/displayRightColumn), footer. É usado para expandir as oportunidades de design e incluir anúncios ou blocos de informação para sua loja. É um módulo com suporte para vários idiomas que pode ser incluído em um site com diversas lojas.

Para incluir o conteúdo, você deve seguir os passos abaixo:

  1. Vá para Módulos (Modules)->Fachada(Front Office Features)->TM HTML Content (se não estiver instalado, faça-o).

    1. Clique em Adicionar novo (Add new).
    2. A lista de idiomas disponíveis. Você pode verificar os elementos disponíveis para todos os idiomas mudando as abas de idioma.
    3. A lista de elementos adicionados para uma posição e idioma específicos.

  2. Adicionar elemento. Clique em Adicionar item(Add item) e preencha as configurações necessários (nenhum dos campos precisam ser preenchidos).

    1. Language - escolha o idioma necessário da lista.
    2. Title - título do elemento.
    3. Use title in front - mostrar o título do elemento no site (será incluído antes do elemento).
    4. Hook - posição onde o elemento será adicionado.
    5. Image - envio de uma imagem do seu computador.
    6. Image width - largura da imagem.
    7. Image height - altura da imagem.
    8. URL - adicione um link (o bloco inteiro será envolvido por este link).
    9. Target blank - abrir link numa aba nova.
    10. HTML - Código HTML (exceto o link se você tiver preenchido o campo de URL acima) ou texto.
    11. Save - clique para salvar o elemento.
    Todos os elementos de um idioma adicionados na mesma posição serão mostrados como uma lista.

  3. Alterando e removendo o elemento

    Quando você adiciona um elemento, ele aparece na lista correspondente.

    1. O idioma onde ele foi adicionado.
    2. Posição onde ele foi adicionadoPosition.
    3. Botão de edição com uma lista.
    4. Botão de apagar (Delete).

     

    Alterando

    1. Clique em Alterar (Edit) para modificar o elemento.

      Na tela que aparece, mude todos os campos necessários.

      1. Enable - mostra o elemento.
      2. Image title - título do elemento.
      3. Use title in front - mostra o título.
      4. Hook to which the image should be attached - posição.
      5. Load your image - imagem.
      6. Image width - largura da imagem.
      7. Image height - altura da imagem.
      8. Target link - link do elemento.
      9. Open link in a new tab/page - abrir em outra aba.
      10. Optional HTML code - alterar o código HTML.
      11. Save - salvar alterações.

  4. Para mudar a ordem dos elementos na lista

    Basta arrastar o elemento para a posição que você precisar.

TM Categories Badges

This module allows to add badges to the sidebar categories of the default Prestashop Categories module ("Blockcategories"). The badges can be added to the category of any level and in any language. The text field of the label is mandatory and should be filled in all languages. If this field is left blank in any language, then it will automatically take the text of the default language.

The module is installed and deleted like any other Prestashop module.

Module Configuration

After installation you can change its settings on the module configuration page.

You can add badges to any category after selecting it from the drop down list. Attention! You can add only one badge for each category. Then, you can add text that will show up in the badge and select the style from the drop-down list (the styles may differ in different templates).

You can see the list of existing badges below. Here you can edit or delete badges with the help of corresponding buttons.

The module is multilanguage and supports the multi-stores. You can add badges for group of stores or all stores at once. They will show up in the list as separate elements.

TM Google Map

This module allows to add Google map with your stores location and short description about them on your front page. The map will show up the addresses of all stores related to your online shop. You can edit your stores details under Preferences->Store Contacts tab.

Installation and Deleting

The module is installed and deleted like any other Prestashop module.

Module configuration

After installation you can change its settings on the module configuration page.

You can configure the style and elements of your map.

Select the map style from the first list (all styles are taken from Snazzymaps). You can also add the style from this website. Find instructions on adding them at the end of this instruction.

Select the view type from the second list (satellite view or street view).

You can also select the zoom ratio for your map from 1 to 17.

Zoom option if scrolling on the map area.

You can disable the map control and ability to view the streets.

The module supports the multi-stores. You can change the map styles for each store.

Adding custom styles

You can add any custom style from Snazzymaps to your module. To do this, you should select the style from the website, copy JavaScript Style Array to .js file and save it in the module folder /js/styles/(for the file name use the style name replacing the spaces with underscores and removing all special characters. For example: for Subtle Grayscale style - subtle_grayscale.js file, for Apple Maps-esque style - apple_maps-esque.js file). Then, navigate to your module configuration page and select the required style from the styles drop-down list.

TM Manufacturers Block

This module shows up manufacturers available in your store. The list of manufacturers may consist of the image and title. The list can also show up as a carousel.

Installing and Deleting

The module is installed and deleted like any other Prestashop module.

Module configuration

After installation you can change its settings on the module configuration page.

  1. Display name - show the manufacturer's name.
  2. Display image - show the image of the manufacturer.
  3. Number of elements to display - select the number of elements displayed in the module.
  4. Image Type - select the image type (you can select the image from the images related to the manufacturer).
  5. Use carousel - show up the list as a carousel.
  6. * Visible items - number of visible manufacturers.
  7. * Items scroll - number of manufacturers shown in one carousel scroll.
  8. * Slide Width - the width of the element in the carousel.
  9. * Slide Margin - the margin of the element in the carousel.
  10. * Auto scroll - automatic scrolling.
  11. * Carousel speed - speed of the carousel.
  12. * Pause - pause between scrolls(if "Auto scroll" is enabled).
  13. * Random - carousel start from the random element.
  14. * Carousel loop - carousel returns back to the beginning when all elements have been scrolled.
  15. * Hide control on end - hide next/previous buttons.
  16. * Pager - show pagination.
  17. * Control - show next/previous buttons.
  18. * Auto control - show play/pause buttons.
  19. * Auto hover - pausing when hovering over carousel elements.

* - settings are available only if carousel is enabled.

The module supports multi-stores.

TM Product Videos

This module allows to add unlimited number of videos to any product in any language and store.

Installation and deleting

The module is installed and deleted like any other Prestashop module. It has no configuration page and supports multi-stores.

After module installation you will see the new tab in your product configuration page (Catalog-> Product-> select any product - TM Product Videos).

To add a video to any product, you should fill in the required fields (the link field is mandatory). The module is multilanguage, so you will have to fill in the fields for all languages this video is related to. If the Link field is not filled for any language, then the video will not be added for this language. The module is supposed to show up videos from Youtube and Vimeo, so you need to add links only from these resources. After making the changes, click Save/Save and Stay button to save them.

Examples of links

To display video properly, the links should have the following format:

  • Youtube - https://www.youtube.com/v/K6VcTyaym90

  • Vimeo - http://player.vimeo.com/video/66140585

If the product has video related to this language, you will see the list of videos below.

To save, delete or edit the list, there is no need to click Save/Save and Stay button. Use the buttons located in the video blocks. To change the order of videos, use drag-and-drop method to change their position. To enable/disable the video, use the button next to video. To edit the videos, fill in the fields with required data and click Update button.

To remove the video from the list, click Remove video button.

While switching languages, only the list of videos related to the language will show up.

To view the video in zoomed format, click Zoom button under the video you want.

Carregando dados de servidores do Google

Alguns países tem problemas em se conectar aos servidores da Google. Isso pode causar problemas em carregar fontes e bibliotecas Javascript através do Google CDN. Para resolver esses problemas, siga esses passos:

Não consigo carregar fontes do Google Fonts:

  1. Abra o arquivo \themes\theme****\header.tpl.

  2. Ache a linha: 

    <link rel="stylesheet" href="http{if Tools::usingSecureMode()}s{/if}://fonts.googleapis.com/css?family=[---nome_da_fonte_aqui---]" type="text/css" media="all" />

  3. Substitua por: 

    <link rel="stylesheet" href="http://fonts.useso.com/css?family=[---nome_da_fonte_aqui---]" type="text/css" media="all" />
